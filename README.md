Source of my personal site.

## Running Locally

```shell
git clone https://github.com/dsoaress/dsoares.me.git
cd dsoares.me && pnpm && pnpm dev
```

## Running tests

```shell
pnpm test:ci
```

- [Next.js](https://nextjs.org)
